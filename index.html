JavaScript `Set` is built in Object

`Set` is a built-in object that allows you to store unique values of any type, whether primitive values or object references. It provides a simple, efficient way to manage a collection of unique elements

See the console to relate more details like:

  • Creating a Set

    You can create a new Set by using the new Set() constructor. You can optionally pass an iterable (like an array) to initialize the set with values.

  • Properties of Set

    The `size` property returns the number of elements in a set

  • Methods

    • add(value): adds the new element with specified value to the `Set`. If the element is already present, it is NOT added again.
    • delete(value): removes the element with specified value from the `Set`. It returns `true` if element present and successfully deleted otherwise `false`.
    • has(value): returns Boolean indicating whether the `Set` contains a specific element
    • clear(): removes all elments from the `Set`
    • foreach(cbFunction, thisArgument): allows you to interate over elements in the `Set`
    • ...and more: `Set` has methods like: `values()`, `keys()`, `entries()` which return iterators over the values, keys, and key-value pairs in the set, respectively.
